# Physical AI & Humanoid Robotics

## Welcome to the Future of Robotics! 🤖

This textbook teaches you how to design, simulate, and deploy 
**humanoid robots** using cutting-edge AI technologies.

## What You Will Learn

- **ROS 2** — Robot Operating System
- **Gazebo & Unity** — Robot Simulation  
- **NVIDIA Isaac** — AI Robot Brain
- **Vision-Language-Action (VLA)** — LLMs + Robotics
- **Humanoid Robot Development** — Full capstone project

## Course Modules

| Module | Topic |
|--------|-------|
| 1 | The Robotic Nervous System (ROS 2) |
| 2 | The Digital Twin (Gazebo & Unity) |

- Basic Python knowledge
- Understanding of AI/ML fundamentals
- Curiosity about robots! 🚀
